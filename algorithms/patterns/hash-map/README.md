# Hash Map Pattern

## Idea
Use a hash map to store frequencies, indices, or grouped data for fast lookup.

---

## When to Use
- Frequency counting
- Anagram problems
- Grouping similar items
- Subarray sum problems

---

## Key Insight
Trade space for time (O(n) speed using O(n) memory)

---

## Complexity
- Time: O(n)
- Space: O(n)
