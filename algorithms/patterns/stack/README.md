# Stack Pattern

## Idea
Use LIFO (Last In First Out) structure to solve problems involving nested or sequential dependencies.

---

## When to Use
- Parentheses validation
- Expression parsing
- Backtracking-like problems
- Monotonic problems

---

## Key Insight
Stack helps manage "last seen / last opened" elements efficiently.

---

## Complexity
- Time: O(n)
- Space: O(n)
