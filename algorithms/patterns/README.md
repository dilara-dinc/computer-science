# Algorithms Patterns

This folder contains core problem-solving patterns used in technical interviews and real-world software engineering.

---

## Purpose

The goal is not to memorize solutions, but to recognize patterns and apply them efficiently.

---

## Core Patterns

- Two Sum
- Sliding Window
- Two Pointers
- Stack
- Binary Search
- Hash Map

---

## Structure

Each pattern contains:
- Problem explanation
- Implementation (Python / Java)
- Key idea and use cases

---

## Goal

To build strong problem-solving intuition for:
- Software engineering interviews
- Backend system logic
- Real-world coding problems
