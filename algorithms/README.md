# Algorithms

This repository is a structured collection of algorithm problems and patterns.

---

## Structure

### Patterns
Core problem-solving techniques used to approach different types of algorithmic questions.

Examples:
- Two Sum
- Hash Map
- Sliding Window
- Two Pointers
- Stack
- Binary Search

These show the general approach used to solve problems.

---

### Problems
Practice problems solved using the patterns above.

These help reinforce how each pattern is applied in real coding situations.

---

## Goal

To improve algorithmic thinking and become more consistent in solving coding problems.
