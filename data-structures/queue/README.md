# Queue

Queue implementation using array-based structure in C++ and Java.

## Purpose

Understanding FIFO (First In First Out) data structure.

## Key Idea

- First element added is the first one removed.

## Operations

- enqueue → O(1)
- dequeue → O(1)
- front → O(1)

## Use Cases

- Task scheduling
- Print queue
- BFS (Breadth First Search)
- Producer-consumer systems
