# Linked List

Singly linked list implementation in C++ and Java.

## Purpose

Understanding dynamic memory and pointer-based data structures.

## Key Idea

Each node stores data and a reference (pointer) to the next node.  
Memory is not contiguous like arrays.

## Complexity

- Insert at head: O(1)
- Insert at tail: O(n)
- Search: O(n)
- Delete: O(n)
