# Tree

Binary Search Tree (BST) implementation in C++ and Java.

## Purpose

Understanding hierarchical data structures and recursive algorithms.

## Key Idea

- Each node has at most 2 children (left, right)
- Left < root < right (BST property)

## Operations

- insert → O(log n) average
- search → O(log n) average
- traversal → O(n)

## Traversals

- Inorder
- Preorder
- Postorder
