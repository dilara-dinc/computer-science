# Stack

Stack implementation using array-based structure in C++ and Java.

## Purpose

Understanding LIFO (Last In First Out) data structure.

## Key Idea

- Last element added is the first one removed.

## Operations

- push → O(1)
- pop → O(1)
- top → O(1)

## Use Cases

- Function call stack
- Undo/Redo operations
- Expression evaluation
- DFS algorithms
